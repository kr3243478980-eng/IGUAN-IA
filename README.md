IGUAN IA - Asistente de IA para Barrancabermeja IGUAN IA es un sistema de Generación Aumentada 
por Recuperación (RAG) diseñado para brindar información precisa sobre las necesidades y datos 
de la ciudad de Barrancabermeja.

Estructura del Proyecto (Beta 0.1) 
 -iguan.py: El núcleo del asistente que procesa las consultas del usuario. 
 -ingesta.py: El script encargado de leer y vectorizar la información local. 
 -datos.txt: La base de conocimientos con información relevante de Barrancabermeja. 
 -.gitignore: Archivo de configuración para mantener el repositorio limpio y seguro.

Tecnologías Utilizadas 
-Python: Lenguaje principal de desarrollo. 
-Ollama & Llama3: Modelos de lenguaje de gran escala (LLM) para el procesamiento natural. 
-RAG (Retrieval-Augmented Generation): Técnica para mejorar las respuestas de la IA con datos locales. 
-WSL2 (Ubuntu): Entorno de desarrollo en Linux.
